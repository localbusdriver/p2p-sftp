# P2P File Transporter
  
This desktop application uses SFTP to transfer files with other "peers" over the same network.  
This application is being developed in:
- ### Frontend:
  - Wails
  - Nextjs
  - React
  - TypeScript
  - Tailwind CSS
- ### Backend
  - Go lang

### Completed Functions:
- User config (username)
  - Store user config
  - Fetch user config
